Polish constitution and Uokik parser
Command line options:

 -f,--file <FILE>                   file path 
 -k                                 information that the parsed document is Consitution
 -r,--range <RANGE>                 display range of articles (e.g. -r 12 43a)
 -s,--specific-elem <ELEM>          display specific document element (e.g. -s art. 1., ust. 2., pkt. 3. or -s dzia³ II, rozdzia³ 1)
 -t,--table-of-contents <SECTION>   display table of contents (e.g. -t or -t II)
